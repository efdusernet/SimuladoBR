Chat Widget Host

Página de demonstração para validar o widget embeddable. Se o botão flutuante aparecer no canto inferior direito, o widget carregou corretamente.

Widget: aguardando…

API configurada: http://localhost:4010
Como testar
  • Suba o serviço: npm run dev
  • Abra esta página e procure o botão do chat no canto
  • Envie uma mensagem pelo widget e veja no painel /admin
Embed usado aqui
<script
  src="http://localhost:4010/widget/chat-widget.js"
  data-chat-api="http://localhost:4010"
  data-chat-title="Suporte"
></script>
Dicas
  • Se o widget não aparecer, verifique /health na API.
  • Se o admin estiver em outro host, configure o API Base no painel e ajuste CORS_ORIGINS.
